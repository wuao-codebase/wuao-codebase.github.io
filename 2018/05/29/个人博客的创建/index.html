个人博客的创建

发表于 2018-05-29 | 分类于 博客 | 热度: ℃
字数统计: 569 | 阅读时长 ≈ 2

本博客使用Gihub+Hexo+next在Windows下创建，贼简单，贼方便，最重要是近乎免费！（非单身狗可以买个域名秀秀恩爱 (╯°Д°)╯︵ ┻━┻,单身狗买个域名装装逼ヾ(≧▽≦*)o。）

Gihub部分

  • 注册Gihub账号，如用户名为“王旭”。贼简单

  • 安装git，贼简单。需注意下图设置，不然就要设置git的环境变量。

    环境变量Path添加：安装目录\bin和安装目录\mingw64\libexec\git-co
    验证成功：右击->Git Base Here->输入 git -version

  • 创建仓库，此仓库也就是我们的微博仓库了。

    仓库名一定设为: 用户名 .github.io！！这也将是我们的微博域名

  • 生成密钥

    1. 首次使用，右击->Git Base Here-> 执行git config --global user.name "xxx"和git config --global user.email "xxx@gmail.com"，设置账户信息。
    2. 输入ssh-keygen -t rsa -C "youremail@example.com"，多次回车到结束。
    3. 打开C:\Users\m1552\.ssh\id_rsa.pub公钥文件，公钥结尾是”youremail@example.com“，复制公钥。
    4. Gihub主页，点击右上角头像->Setting->SSH and GPG keys，添加SSH keys。
    5. 右击->Git Base Here->输入ssh git@github.com，提示You’ve successfully authenticated。成功。

Hexo部分

  • 安装Node.js

    使用node -v和npm -v检测是否安装成功。

  • 安装Hexo

    1. 创建E:\blog文件夹
    2. 在文件夹中右击->Git Base Here->输入 npm install -g hexo-cli,下载完成
    3. 执行hexo init和npm install完成安装。
    4. 执行hexo -v检验版本

    5. 打开E:\blog\_config.yml文件,进行如下设置将Hexo和github联系起来。

    6. 执行hexo new "第一篇文章"，创建文章(E:\blog\source_posts\第一篇文章.md)

    7. 执行hexo g&hexo s启动本地服务，默认端口4000
    8. 浏览器打开http://localhost:4000，访问成功结束。
    9. 访问失败，说明端口被占用。“crtl+c”结束服务，执行hexo s -p 5000更换端口,直到看到博客页面。

博客的部署

本地调试成功后，执行hexo d -g进行生成部署博客。访问https://用户名.github.io/成功。个人博客初始化搭建完成，但是很简陋不利于我们装逼用，接下来将进行置和美化。
