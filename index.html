在IDEA中简易搭建搭建Spring-boot 2.0.3开发环境

1. 准备

  • 安装IDEA
    在settings -> Plugins 里面搜索spring boot，勾选上.
  • 安装Maven
  • 安装JDK

2. 创建spring boot项目

选择NEW -> Project -> Spring Initializr

填写项目名和包名。

选择组件自动生成依赖，这里直选web组件

选择项目路径，完成创建。

3. 初始化项目文件

3.1 pom.xml

IDEA已经根据我们的选择添加了一些依赖，这里只用添加configuration-processor依赖，这是2.0版本后要求的读取自定义配置所用的依赖包。

3.2 application.yml

spring boot项目的主配置文件，配置数据库，日志，连接池等组件功能都在这里配置。先简单使用下。

3.4 Student.java

bean类，这里用来存储自定义对象

3.3 HelloController.java

Controller类，对请求的url进行控制。

3.4 SpringbootApplication.java

spring-boot的主程序，已自动生成，在这里不用做过多配置。运行主程序启动spring-boot，访问localhost:8080/test/hello。
